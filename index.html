Instructions: In this activity you are creating a command-line calculator application. When completed, the app should perform the following:

    1. Ask the user what kind of arithmetic operation they would like to use.

    2. Add two integers together if the user selects addition.

    3. Subtract two integers from one another if the user selects subtraction.

    4. Multiply two integers together if the user selects multiplication.

    5. Divide two integers by one another if the user select division.

    6. Ask the user if they'd like to play again. If yes, go back to asking what kind of operation they'd like to perform. If no, exit with a message saying "Thank you for playing!"

    7. Each of the arithmetic operations should be contained within functions that take in two integers as parameters.
